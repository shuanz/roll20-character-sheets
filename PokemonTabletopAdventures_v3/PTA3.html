Configuration Options

Character Type: ℹ

This setting will change which options are available to use in the 'Character' page.
The Pokémon Character Class option is best used to represent the Pokémon class from the PHB2, but can also be used for other things, and is especially suited to custom classes.

Initiative Tie Breaker: ℹ

This setting is not used by the character sheet, but is incldued so that you can, if you wish, add it to the character's speed when determining their initiative order. (Note this isn't an official rule, but it's helpful when characters have the same speed!)
For best results, leave this as something you can add to a die-roll.

Type & Background Color: ℹ

This setting will update the first type for Pokémon-type characters, and also change the background color and theme for all types of characters.
